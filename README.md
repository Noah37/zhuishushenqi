## Requirements

- iOS 9.0+ / macOS 10.14+ / tvOS 9.0+ / watchOS 2.0+
- Xcode 10.2+
- Swift 5.0+

Main development of zhuishushenqi olny support Swift 5.0+.

## Install

请执行以下两步操作，确保所有的依赖库安装完成后使用.

**CocoaPods:**

`$ pod install`

**Carthage:**

`$ carthage update --platform iOS`
